# PAMA Proje Mühendislik İnşaat Web Sitesi

Bu paket, PAMA tanıtım sunumuna dayalı hazırlanmış ultra premium statik web sitesi kodlarını içerir.

## Dosya yapısı

```text
pama_web/
├── index.html
├── assets/
│   ├── css/styles.css
│   ├── js/main.js
│   └── images/
│       ├── pama-logo.png
│       ├── pama-mark.png
│       ├── *.jpg
│       └── *.webp
├── robots.txt
├── sitemap.xml
└── site.webmanifest
```

## Yayına alma

1. `pama_web` klasöründeki tüm dosyaları hosting hesabınızda `public_html` veya domain kök dizinine yükleyin.
2. Domain kökü `https://www.pama.com.tr/` olacak şekilde DNS/hosting yönlendirmesini yapın.
3. İletişim e-postasını değiştirmek için `index.html` ve `assets/js/main.js` içinde `info@pama.com.tr` araması yapın.
4. İletişim formunu gerçek kayıt sistemiyle çalıştırmak için formu PHP, Node.js, ASP.NET veya kullandığınız CRM/API altyapısına bağlayın.

## İçerik notu

Referans ve ekipman başlıkları, sağlanan tanıtım sunumundaki içeriğe göre web formatına dönüştürülmüştür. Yayından önce firma tarafından son doğrulama yapılması önerilir.
